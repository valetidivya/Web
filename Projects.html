Automatic Toll Collection By Using RFID

Aim : In this project, we will introduce an Automatic Smart Toll Tax
        Collection System designed using RFID Module and Arduino.
        Out come: The RFID Readers mounted at toll booth will read the
        prepaid RFID tags fixed on vehicles windshield and automatically
        respective amount will be deducted.
